A basic etch-a-sketch created using vanilla Javascript.

<h2>Features:</h2>
- The board defaults to a 16x16 grid
- User can customise the grid size by clicking the 'Set grid size' button. Input has basic validation.
- User can clear their drawing by pressing 'Reset'
- User can toggle eraser mode by clicking the 'Draw mode' button (the button text shows the currently active move)
